# TestNG VSCode

Maven-based TestNG automation framework configured for VS Code development.

## Overview

Java automation testing framework using TestNG with Maven build configuration.

## Project Structure

- **src/** - Source code and test files
- **pom.xml** - Maven configuration with TestNG dependencies
- **.github/** - GitHub Actions CI/CD workflows
- **.vscode/** - VS Code workspace settings
- **.gitignore** - Git ignore patterns

## Technology Stack

- Java
- TestNG (testing framework)
- Maven (build tool)
- VS Code (IDE)

## Setup

1. Build the project:
   ```bash
   mvn clean compile
   ```

2. Run tests:
   ```bash
   mvn test
   ```

## Features

- CI/CD integration via GitHub Actions
- VS Code workspace optimization
- Maven dependency management
